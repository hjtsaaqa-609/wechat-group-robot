# 经营周报 KPI 改版说明

本次改版将 `DAS周报` 从“运营明细周报”调整为“整体经营 + 各部门 KPI”格式。

当前纳入周报的区块：

- 公司整体：机器人累计上线、确认收入、公司整体盈利
- 客户满意度：年度累计投诉数量、客户满意度
- 市场销售部：机器人累计上线、确认收入
- 生产部：出厂良品率、发货正确率、季度库存盘点准确率、单台机器人制造成本
- 研发部：PCR-300 新产品批量生产、PCR-300 量产成本控制、CTR 工程样机交付、每 GW 远程接管次数、每 GW 现场维护次数、年度累计每 GW 毛刷更换次数
- 运营部：每 GW 远程接管次数、每 GW 现场维护次数、新增 3 级故障数、清洁度未达预期率、年度累计异常组件解决率、发电量未达预期率、屋顶形变故障沟通率
- 项目部-费用控制目标：初步方案设计费、详细方案设计费、安装调试费、年化耗材更换费用、年化故障处理费用
- 项目部-质量目标：安装一次成功率、运维成功率
- 项目部-时效目标：毛刷老化待更换、毛刷老化平均持续时间、待处理 2 级故障、待处理 2 级故障平均持续时间

当前统一口径：

- 机器人累计上线：`试运行 + 正式运营`
- 数据统计口径：以 DAS 当前接口为准
- 客户投诉：按年度累计统计
- 异常组件解决率：改为按年度累计统计；若 DAS `inspection-stats` 年累计区间不可稳定查询，则以 `尚未入系统` 标识
- 每 GW 毛刷更换次数：改为按年度累计统计，优先使用 `fault` 模式统计结果
- 清洁度未达预期率：采用 DAS 清扫质量页面的电站维度 `belowExpectedRatio`
- 项目部时效目标：暂以旧运营模板中的 backlog/持续时长指标代理展示
- `尚未入系统`：目标已在《织光智能2026年经营目标分解-For AI.pptx》中明确，但当前 DAS 缺少稳定字段或统计口径仍待确认

当前已自动接入的指标：

- 机器人累计上线
- 年度累计投诉数量
- 出厂良品率、发货正确率
- 每 GW 远程接管次数、每 GW 现场维护次数、年度累计每 GW 毛刷更换次数
- 新增 3 级故障数、年度累计异常组件解决率、发电量未达预期率
- 安装一次成功率、运维成功率
- 清扫质量平均持续时间、待处理 2 级故障平均持续时间

当前以 `尚未入系统` 标识的指标：

- 确认收入、公司整体盈利
- 客户满意度
- 季度库存盘点准确率、单台机器人制造成本
- PCR-300 新产品批量生产、PCR-300 量产成本控制、CTR 工程样机交付
- 屋顶形变故障沟通率
- 初步方案设计费、详细方案设计费、安装调试费、年化耗材更换费用、年化故障处理费用
