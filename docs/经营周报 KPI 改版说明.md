# 经营周报 KPI 改版说明

本次改版将 `DAS周报` 从“运营明细周报”调整为“整体经营 + 各部门 KPI”格式。

当前纳入周报的区块：

- 公司整体：机器人累计上线
- 客户满意度：年度累计投诉数量
- 市场销售部：机器人累计上线
- 生产部：出厂良品率、发货正确率
- 研发部：每 GW 远程接管次数、每 GW 现场维护次数、每 GW 毛刷更换次数
- 运营部：新增 3 级故障数、异常组件解决率、发电量未达预期率
- 项目部-质量目标：安装一次成功率、运维成功率
- 项目部-时效目标：毛刷老化待更换、毛刷老化平均持续时间、待处理 2 级故障、待处理 2 级故障平均持续时间

当前统一口径：

- 机器人累计上线：`试运行 + 正式运营`
- 数据统计口径：以 DAS 当前接口为准
- 客户投诉：按年度累计统计
- 项目部时效目标：暂以旧运营模板中的 backlog/持续时长指标代理展示

本期未纳入自动周报的指标：

- 财务相关指标
- 库存准确率、制造成本
- 研发里程碑类时效指标
- 屋顶形变故障沟通率、清洁度未达预期率
- 原 PPT 中的安装交付时效与故障分级处理时效
